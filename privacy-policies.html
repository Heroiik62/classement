Politique de Confidentialité – Drone

Dernière mise à jour : 5 mai 2025

1. Informations collectées

Le bot Drone peut collecter les informations suivantes lors de son utilisation :

  • Identifiants Discord des utilisateurs (user ID)
  • Identifiants de serveurs (guild ID)
  • Informations liées à l'utilisation des commandes (ex : fréquence, paramètres utilisés)

Aucune donnée sensible (comme les messages privés ou informations personnelles) n’est collectée.

2. Utilisation des données

Les données sont utilisées uniquement pour :

  • Faire fonctionner les fonctionnalités du bot
  • Améliorer l’expérience utilisateur
  • Gérer d’éventuelles fonctionnalités premium

3. Partage des données

Aucune donnée n’est partagée avec des tiers. Les données restent strictement internes à l'application Drone.

4. Stockage des données

Les données sont stockées de manière sécurisée sur des systèmes contrôlés par le développeur.
Le développeur est un particulier indépendant, et bien que tous les efforts soient faits pour protéger les données, aucune garantie absolue ne peut être donnée.

5. Droits des utilisateurs

Si vous souhaitez que vos données soient supprimées ou si vous avez une question liée à la vie privée, vous pouvez contacter le développeur via Discord à l’adresse suivante : @heroiik62

6. Consentement

En utilisant le bot Drone, vous acceptez cette politique de confidentialité.
